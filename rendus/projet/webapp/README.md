# DevOps CV Webapp

Voir le [README principal](../README.md) pour la documentation complète du projet.

## Démarrage rapide

```bash
npm install
docker run -d -p 6379:6379 redis:7-alpine
npm start
```

- CV : http://localhost:3000
- Swagger : http://localhost:3000/api-docs
- Health : http://localhost:3000/health
